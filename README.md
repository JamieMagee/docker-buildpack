# docker-buildpack

This repository builds the image `renovate/buildpack` on Docker Hub.

The image is used as a centralized build source for:
- [x] `renovate/composer`
- [ ] `renovate/go`
- [ ] `renovate/gradle`
- [ ] `renovate/npm`
- [ ] `renovate/pipenv`
- [ ] `renovate/pnpm`
- [ ] `renovate/poetry`
- [ ] `renovate/ruby`
- [ ] `renovate/yarn`
